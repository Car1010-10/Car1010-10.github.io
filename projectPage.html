Project

"Catabasis"

This project took me roughly 3 months to complete. I also worked with one other person on this project. We used the Unity Engine to complete this project.

This project is a top down game that has gameplay inspired from the Hotline Miami series.

On this project I worked on implementing player movement, a health system used by both player and bots, and a attack system. That attack system consisted a ranged projectile attack and a melee attack, used by both player and bots.
